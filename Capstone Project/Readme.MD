#Foodie – Food Delivery App (Frontend Project)

##A modern and responsive food delivery landing page designed using HTML5, CSS3 (Flexbox & Grid), and UI/UX best practices.

This project aims to showcase a food delivery service called Foodie, enabling users to browse food categories, explore restaurants, view promo offers, and read customer reviews — all within a clean and visually rich interface.
Features

Fully responsive layout (Mobile, Tablet & Desktop)
Clean UI with modern visual design
Built using CSS Grid and Flexbox
Semantic and accessible HTML elements
Promo offer display with coupon codes
Customer testimonials with avatars
Browse menu categories & featured restaurants
Smooth hover animations and subtle micro-interactions
Social media icons linked in the footer

##Tech Stack

HTML5	
CSS3
Google Fonts
