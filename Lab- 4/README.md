# A simple JS quiz game
It is a simple JavaScript-based quiz game that runs entirely in the browser console.  
The user answers 10 general knowledge questions, and the quiz provides instant feedback along with a final score summary.

---

## Features

- Runs in browser console
- Shows quiz questions using `prompt()`
- Checks answers using string methods (`toLowerCase()` + `trim()`)
- Uses arrays, loops, and conditionals for quiz logic
- Tracks score throughout the quiz
- Displays final feedback message using a `switch-case` structure
- Simple and user-friendly alerts

## How to Run the Quiz

1. Open **Google Chrome** or any browser
2. Press 'ctrl+shift+i' to open **Developer Tools**
3. Go to the **Console** tab
4. Copy all code from `quiz.js'
5. Paste it into the console and press **Enter**
6. Answer the prompts and view your score at the end!
