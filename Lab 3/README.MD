# Personal Portfolio Website

This is a responsive personal portfolio webpage created as part of the Web Development Lab assignment.  
It showcases the hero section, navigation bar, skills, projects, and contact form — with a modern and visually appealing design.

---

##  Lab Title
**Lab – Responsive Portfolio Webpage Development**

##  Description
The objective of this lab is to design and develop a personal portfolio website using **HTML5** and **CSS3**, applying:
- Responsive layouts using **Flexbox**
- Semantic HTML structure
- Modern styling practices with hover transitions and custom color theme
- Media queries to ensure cross-device support


---

## Viewing Responsive Behavior

The website has been tested across different screen sizes using browser DevTools.  
You can check the responsive behavior in the following modes:

| Desktop / Laptop | Above **1024px** | Full layout with hero image and text side-by-side |
| Tablet | **768px – 1024px** | Hero image centered and layout reflows neatly |
| Mobile (Portrait) | **320px – 768px** | Navigation and hero stack vertically, text becomes center-aligned |
